
# Blink your backlight keyboard.
### steps to do this: 

1) Fire up your terminal by pressing ctrl+alt+t
2) clone this repository 
```bash
git clone https://github.com/pavansrinivasmamidala/blinking_keyboard
```
3) change permissions 
```bash
chmod +x blink
```
4) go into root (sudo su or su)
```bash
sudo su 
```
5) run the script 
```bash
./blink.sh
```
6) Only run this in root as we need administrator access to change the brightness level of the keyboard!
